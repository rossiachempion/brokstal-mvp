# MVP-веб-интерфейс управления цифровым имиджем ООО «Броксталь»

Прототип подготовлен для второй главы отчёта по преддипломной практике. Сервис демонстрирует единый интерфейс мониторинга отзывов, обращений и репутационных событий из VK, Яндекс Карт, 2ГИС и сайта компании.

## Запуск

```bash
python -m venv .venv
.venv\Scripts\activate  # Windows
pip install -r requirements.txt
python app.py
```

После запуска откройте в браузере:

```text
http://127.0.0.1:5000
```

## Состав прототипа

- `app.py` — Flask-приложение, маршруты и работа с SQLite.
- `templates/` — HTML-шаблоны интерфейса.
- `static/styles.css` — оформление веб-интерфейса.
- `static/app.js` — фильтрация обращений на стороне клиента.
- `brokstal_reputation.db` создаётся автоматически при первом запуске.

## Основные функции

- сводная панель показателей;
- список репутационных событий;
- фильтрация по источнику, тональности и статусу;
- карточка обращения с подготовкой ответа;
- изменение статуса обращения;
- добавление тестового обращения;
- аналитическая страница по источникам и тональности.
